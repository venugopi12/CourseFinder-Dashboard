# CourseFinder-Dashboard
Project Overview

This project is a CourseFinder-Dashboard User Interface built using HTML, CSS, and JavaScript.
The dashboard is created based on a Figma design and includes:
Student management
Event registration
Dashboard cards
Forms with validation
Modal popup handling
This is a frontend-only project (no backend).

What This Project Does :
Shows a dashboard layout with header, sidebar, and content area
Displays statistics, applications, events, and contacts

Allows users to:
Register a Student
Register for an Event
Uses a single reusable form modal for both Student and Event registration
Validates form inputs before submission
Shows a success message after successful form submission

Technologies Used :
HTML5
CSS3
JavaScript (Vanilla JS)
Bootstrap Icons
Google Font: DM Sans

Page Layout Explanation Header Fixed at the top Contains:
Logo,Search bar,Notification icon,User profile

Main Content Area Includes:
Greeting message, Filters section, Statistics cards, Applications table, Allied Services banner, News bulletin, Upcoming events card, Quick links, Contact Regional Manager section


This project demonstrates:
Clean UI implementation
Figma to HTML conversion
Form validation using JavaScript
Reusable components
Responsive layout design
